# FDUer-Manual

&lt;复旦大学生存手册&gt;由24年一群复旦本科和研究生写成，本着希望前人走过的泥泞能给后人趟出一条经验之路的想法，在各位学长学姐欢迎所有希望分享经验的复旦学子参与到手册的编写中来。无论是关于学术研究的心得体会，还是有关职业发展的独到见解，我们都期待您的贡献。此外，我们也鼓励非复旦大学的学生们分享有关实习、转学等领域的经验。希望这些抽空写的内容能给后来人哪怕一点微小的帮助，这里感谢所有的作者对这个项目的贡献！



欢迎所有希望分享经验的复旦学子参与到手册的编写中来。无论是关于学术研究的心得体会，还是有关职业发展的独到见解，我们都期待您的贡献。此外，我们也鼓励非复旦大学的学生们分享有关实习、转学等领域的经验。
